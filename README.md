# api-students
API to students 

Routes: 
- Get /students - List all students
- Post / students - Create students
- Get /students/:id - Get info from a specific student 
- PUT /students/:id - Update students
- DELETE /students/:id - Delete students

Struct Students:
- Name (string)
- CPF (int)
- Email (string)
- Age (int)
- Active (bool)
